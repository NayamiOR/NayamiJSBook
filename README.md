# 简介

Js曾经主要用于前端的操作比如参与页面的显示，但是node.js之后Js可以用来做后端程序了，也就是说Js从此以后可以前端后端一把抓（全栈）。
## 学习顺序

1. DOM
2. BOM
3. 异步，Ajax

# 基础语法

## 类型注解

### 传参

```typescript
function greeter(person: string) {
    return "Hello, " + person;
}
```

### 返回值

```typescript
function greeter(person: string): string {
    return "Hello, " + person;
}
```

### 变量

```typescript
let isDone: boolean = false;
```

### 数组

```typescript
let list: number[] = [1, 2, 3];
```
